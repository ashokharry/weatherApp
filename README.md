Built With

* Java 8 - Code Language
* Spring Boot 2 - Web Framework used
* Maven - Build Tool

Testing
* Junit
* WireMock

Using an IDE
1. git clone https://github.com/ashokharry/weatherApp.git
2. import as existing project
3. run as spring boot project

Using Command Line
1. git clone https://github.com/ashokharry/weatherApp.git
2. cd weatherApp
3. mvn clean install
4. mvn spring-boot:run
